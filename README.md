# Rock Paper Scissors Game

A simple and interactive Rock Paper Scissors game built using HTML, CSS, and JavaScript.

## Features

* Rock Paper Scissors gameplay
* Score tracking
* Dark Mode
* Responsive Design
* Reset functionality

## Technologies Used

* HTML5
* CSS3
* JavaScript
